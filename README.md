## 🧩 Generación de Patrones ASCII

El usuario puede generar diferentes figuras como:

- Triángulos
- Cuadrados
- Pirámides
- Otros patrones geométricos

A continuación se describen las diferentes funcionalidades implementadas dentro del programa.

## Patrones ASCII

### Triángulo

Genera un triángulo utilizando caracteres `*`.  
El tamaño del triángulo depende del número ingresado por el usuario.

Ejemplo:

*
**
***
---

### Cuadrado

Crea un cuadrado utilizando caracteres ASCII.

Ejemplo:

* * * *
*     *
*     *
* * * *

---

### Pirámide

Genera una pirámide centrada utilizando caracteres `*`.

Ejemplo:

    *
   ***
  *****
 *******
*********

---

## Texto Artístico

Esta opción permite generar texto decorativo dentro de un marco ASCII.

Ejemplo:

********
* Axel *
********

---

## Animaciones

El programa incluye pequeñas animaciones que se ejecutan directamente en la terminal mediante impresión dinámica de caracteres.

### Barra de progreso

Simula el progreso de una tarea mediante una barra animada.

Ejemplo:

[####------] 40%
[######----] 60%
[########--] 80%
[##########] 100%

---

### Texto en movimiento

Un texto se desplaza horizontalmente en la terminal para crear un efecto visual simple.
