## 🧩 Generación de Patrones ASCII

El usuario puede generar diferentes figuras como:

- Triángulos
- Cuadrados
- Pirámides
- Otros patrones geométricos

A continuación se describen las diferentes funcionalidades implementadas dentro del programa.

## Patrones ASCII

### Triángulo

Genera un triángulo utilizando caracteres `*`.  
El tamaño del triángulo depende del número ingresado por el usuario.

## Animaciones

El programa incluye pequeñas animaciones que se ejecutan directamente en la terminal mediante impresión dinámica de caracteres.

### Barra de progreso

Simula el progreso de una tarea mediante una barra animada.

Ejemplo:

[####------] 40%
[######----] 60%
[########--] 80%
[##########] 100%

# 📖 Instrucciones de Uso

A continuación se describen los pasos necesarios para ejecutar y utilizar el programa de arte ASCII.

---

## 1. Ejecutar el programa

1. Abrir una terminal o consola en la carpeta del proyecto.
2. Ejecutar el programa

---

## ## 2. Navegar por el menú principal

El menú principal permite acceder a las distintas funcionalidades del programa:

Ejemplo de Menú:

![Menú principal](Imagenes/Menu%20principal.png)

Para seleccionar una opción, el usuario debe escribir el número correspondiente y presionar Enter

---

## 3. Generar patrones ASCII

Si el usuario selecciona la opción de patrones:

1. El programa pedirá el tamaño de la figura.
2. Después generará el patrón en la terminal.

---

## 4. Crear texto artístico
El usuario puede escribir un texto que será mostrado en un formato decorativo utilizando caracteres ASCII.

---

## 5. Ejecutar animaciones

El programa incluye animaciones simples que se muestran directamente en la terminal.

Por ejemplo:

1. Barra de progreso
2. Texto en movimiento

Las animaciones se ejecutan automáticamente después de seleccionar la opción correspondiente.

## 6. Guardar en la galería

Algunas opciones permiten guardar el arte generado en un archivo .txt
Las creaciones se almacenan dentro de la carpeta:

/datos

Esto permite consultar posteriormente las creaciones del usuario

---

## 7. Salir del programa

Para cerrar el programa, el usuario debe seleccionar la opción Salir en el menú principal.
